# cnwebs-cns-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不针对任何特定域名或网站，仅供浏览与参考使用。

## 项目简介

cnwebs-cns-guide 是一个非商业化的归档项目，旨在保存并展示一批独立的 HTML 文档。项目本身不涉及具体网站运营或内容更新承诺，仅作为静态页面的发布与存储地。

## 目录说明

```
.
├── index.html          # 项目入口页面（如有）
├── pages/              # 存放各独立 HTML 页面
├── assets/             # 页面引用的静态资源（图片、样式、脚本等）
└── README.md           # 本文件
```

- `pages/` 目录下包含多个独立的 HTML 文件，每个文件为一个完整的页面。
- `assets/` 目录用于存放页面所需的公共或私有资源，按需组织。

## 页面归档说明

- 所有页面均为独立 HTML，不依赖后端服务。
- 页面内容可能随时间更新或调整，但本仓库不保证内容的时效性。
- 归档仅用于信息保存与展示，不构成对页面内容准确性的承诺。
